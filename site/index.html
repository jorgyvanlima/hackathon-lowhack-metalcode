Apresentação

Somos a equipe Metal CODE, participantes do Hackathon LOW CODE 2026. Este é um projeto de apresentação inicial para demonstrar nossa presença e capacidades com arquitetura 100% em Docker e deploy via GitHub Actions.

Membros:

  • Jorgyvan Lima
  • Gusttavo Papini
  • Bruno Santos

Como funciona

O projeto é empacotado em Docker. O GitHub Actions faz deploy ao servidor via SSH, faz git pull e roda docker compose up -d --build, atualizando containers automaticamente.
